Projects


Genreprompt

This is a small mobile-first web application designed to help writers out who are struggling with ideas for their next project. The user can select a genre of choice from the list (or leave it as ‘random’) and the site will randomly generate a set of story ideas from a collection of arrays, fitting them into a template. The user also has the option to take notes on their idea in a text-field and can save said notes into a .txt file using the button provided. The project is built in HTML, CSS and React, and uses Materialize for styling.

This project is live! (Link here) (Hosted on Heroku)

Writers' Notepad

This is a simple full-stack project designed to help writers take notes during Nanowrimo (or other writing events.) Each note can be assigned a category and the app will automatically sort the projects by said category. There are also ‘default’ categories to select, such as ‘world building,’ ‘characters’ and ‘locations.’ Notes are stored in a PostgreSQL database which communicates with Django. This app was built with HTML, CSS, React, Django, PostgreSQL and Bootstrap. This project is not live, but you can download it from github.com/43232
