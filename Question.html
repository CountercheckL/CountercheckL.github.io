Projects


Genreprompt

This is a mobile-first web application designed to help writers out who are struggling with ideas for their next project. The user can select a genre of choice from the list (or leave it as ‘random’) and the site will randomly generate a set of story ideas from a collection of arrays, fitting them into a template. I also added a button that lets the user save their own notes into a .txt file.

The project is built in HTML, CSS and React, and uses Materialize for styling. (Hosted on Heroku.)


Sparrow Environmental Ltd

This is a company website designed for a waste management organisation based in London. It is a two-page site, and was designed with HTML, CSS, React and the Materialize framework.
